Class (클래스의 정의 방법)
-----
- 클래스에 대한 멤버변수와 멤버메서드를 정의한다.
# 3부분으로 나누어지며 클래스명, 변수, 메서드를 나타낸다.
1. 상단 : 클래스명
2. 중단 : 멤버변수
  - - : private
  - + : public
  - # : Pprotected 
3. 하단 : 메서드
```C
------------------------
|-Name
|+Health
|#AttackPower
|---------------
|-Move()
|+Attack()
---------------------------
```
